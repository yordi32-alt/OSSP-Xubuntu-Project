## Project Title

Installation and Study of Xubuntu Operating System in Virtual Environment

---

## Project Summary

This project focuses on the installation of Xubuntu 24.04 operating system using Oracle VM VirtualBox in a Windows 11 environment.

The project also includes:

* Virtualization in modern operating systems
* UNIX standardization and implementation
* Linux filesystem support
* System calls
* Shell scripting for problem solving
* Installation challenges and solutions

The purpose of this project is to understand practical operating system concepts and improve Linux system administration skills.
